# bonesclinic2

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.4. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
## About

This is a Fracture Clinic API designed for orthopaedic clinics to manage patient records, fracture cases, and clinical data.
